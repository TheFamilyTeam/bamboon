# Bamboon
A simple antiflood for telegram, created in python

# How to use:
Simply put your token in the script and run it.

# Dependencies
[Telepot](https://github.com/nickoala/telepot)

# Features
* silent
* sultithreaded
* slim

# Installation
```$ pip install telepot
$ git clone https://github.com/TheFamilyTeam/bamboon```
